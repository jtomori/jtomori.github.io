**Fussel** [ <- ](..) _2022-07-08_ Hey, **Fussel** is a really cool animated movie I've helped with during my studies at Filmakademie. It's been recently publicly released after a festival run so I'd like to share it and note down some of the things I've worked on. ![](https://youtu.be/UdKKnsIMOgI) Btw it's been 3 years since I've worked on it so hopefully I remember things right. This project has had an interesting pipeline, where Alex would sculpt environments and make animation loops in Quill in VR. I helped with post-processing of Quill's exports, to make it pipeline and rendering friendly. Pipeline was done by Tim. Here are some of the steps: - Geometry was cleaned up - Vertex colors were smoothed on some objects to remove the sharp "brush" transitions - Hand-animated loops in Quill had jumps, so a smooth transition was added - Instancing was detected and animation was applied to all instances - Alex would draw some background element, then instance it around the scene and then animate a "hero" instance - Animation from the "hero" instance would be propagated to all other instances, with time offsets to avoid visual repetition - Export scene hierarchy as an Alembic, to be rendered in Maya with Arnold and Multiverse - I think that I also made some string deformation asset, that was used in a Maya rig, via Houdini Engine The processing was done in background, triggered by Tim's UI for ingesting Quill scenes. Processing was automatic, without any user input. It relied on some naming conventions agreed by Alex though, to specify what to do with various objects. Overall it was a very enjoyable project, both from the technical and team work point of views.
